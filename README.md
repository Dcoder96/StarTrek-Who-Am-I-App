### Prérequis
* [NodeJS](https://nodejs.org/)
* [Ionic 3](ionicframework.com/)

### Installation

```bash
git clone https://github.com/Dcoder96/StarTrek-Who-Am-I-App.git
cd StarTrek-Who-Am-I-App
npm i
```

### Lancement

`ionic serve -o`
